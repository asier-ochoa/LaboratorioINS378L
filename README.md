## Como compilar
Requisitos:
- cmake
- git
- Compilador C++ 17

### Comandos:
```
git clone --recursive https://github.com/asier-ochoa/LaboratorioINS378L
cd LaboratorioINS378L
cmake . -B=build
```
Lueago dependiendo del generador que use:
```
-(En linux con make)
cd build && make
-(En windows con mingw64)
cd build
mingw32-make
```
